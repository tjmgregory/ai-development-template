## Contributing

This project uses the [Unified Process](https://en.wikipedia.org/wiki/Unified_process) and
[beads](https://github.com/steveyegge/beads) for development workflow.

### For AI Agents

Read `AGENTS.md` before starting work. It covers:

- The Unified Process and its phases
- How to use beads for issue tracking
- Where to find specs and requirements

### For Humans

1. Check `docs/process/README.md` for current project status
2. Use `bd ready` to see available work
3. Claim issues with `bd update <id> --status in_progress` before starting
4. Follow existing specs; update them if reality diverges

### Claude Code Users

`claude plugin install beads` - see [beads plugin](https://github.com/steveyegge/beads/tree/main/claude-plugin)

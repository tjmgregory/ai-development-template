## Development

This project follows the [AI Unified Process (AIUP)](docs/aiup/README.md) methodology.

- **AI agents**: Read `AGENTS.md` at the start of each session
- **Process docs**: See `docs/aiup/` for requirements, specifications, and project status

## Contributing

This project uses [AIUP](https://aiup.dev/) and [beads](https://github.com/steveyegge/beads)
for development workflow.

### For AI Agents

Read `AGENTS.md` before starting work. It covers:

- AIUP process and which phase we're in
- How to use beads for issue tracking
- Where to find specs and requirements

### For Humans

1. Check `docs/aiup/README.md` for current project status
2. Use `bd ready` to see available work
3. Claim issues with `bd update <id> --status in_progress` before starting
4. Follow existing specs; update them if reality diverges

### Claude Code Users

Install the [beads plugin](https://github.com/steveyegge/beads/tree/main/claude-plugin)
for better issue tracking integration.
